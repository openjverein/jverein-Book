# Lesefelder

Zur Nutzung der Lesefelder ist die Option in den Einstellungen->Administration->Einstellungen zu aktivieren.

Anschließend sollte JVerein neu gestartet werden, damit der Tab "Lesefelder" zur Verfügung steht.
